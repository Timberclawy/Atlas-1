# Atlas-1

## Project Description
Atlas-1 is a 2D top-down dungeon exploration game developed using the Godot engine. Players will navigate through procedurally generated dungeons, battling monsters, and collecting treasures.

## Setup Instructions
To set up the project locally, follow these steps:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/<your_username>/atlas-1.git
   cd atlas-1
   ```

2. **Open the Project in Godot**
   - Launch the Godot Engine.
   - Click on "Open Project".
   - Navigate to the cloned directory and select it.

3. **Install Dependencies**
   If your project requires specific dependencies, ensure they are installed as per the project requirements.

4. **Run the Project**
   - Click on the "Play" button to run the game.

Enjoy exploring the dungeons!
